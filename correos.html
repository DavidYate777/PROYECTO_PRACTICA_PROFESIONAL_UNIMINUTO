Guía de correos

Referencia rápida para saber cuándo usar cada tipo de correo.

Confirmación de solicitud

Se usa cuando el elemento solicitado está disponible en el horario indicado.

  • Para niñeras: indicar que el profesor debe ir al piso 8° a reclamarla.
  • Para laboratorios: indicar que debe usar el QR de la puerta y escribir por Teams.
  • Siempre mencionar que los elementos se revisan antes y después del uso.

Negación por horario no disponible

Se usa cuando el elemento ya está reservado o no hay disponibilidad en el horario pedido. El correo informa claramente el elemento, fecha y hora no disponible para que el docente busque otra opción.

Negación por inconsistencia de horario

Se usa cuando el formulario del docente tiene un error lógico: la hora de finalización es anterior a la hora de inicio. El sistema lo detecta automáticamente y activa una advertencia.

Cierre de turno

Se envía obligatoriamente al finalizar cada turno (tarde o noche).

  • Sin novedad: correo simple indicando que todo quedó en orden.
  • Con novedades: lista los elementos instalados, equipos dejados, pendientes y observaciones importantes para el siguiente turno.
